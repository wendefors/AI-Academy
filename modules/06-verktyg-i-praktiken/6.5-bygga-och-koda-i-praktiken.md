# Bygga och koda i praktiken

Den här kategorin är inte bara för utvecklare. Den är för alla som vill bygga något: ett script som automatiserar en uppgift, ett litet verktyg, en prototyp.

Med AI i kodmiljön sänks tröskeln markant. Du behöver fortfarande förstå vad du bygger och kunna bedöma om det fungerar. Du behöver inte hålla syntaxen i huvudet.

---

## Vad du kan bygga

**Scripts för automation.**
Gör du något manuellt varje vecka som egentligen följer ett mönster? Hämta filer, döp om dem, flytta innehåll mellan format, skicka standardiserade mejl. Det är script-arbete, och AI kan skriva det åt dig om du beskriver vad du vill göra.

**Enkla dataanalyser.**
Har du ett kalkylblad med data du vill analysera på ett sätt som Excel inte hanterar lätt? Beskriv vad du vill se och be AI skriva ett kort Python-script som gör det.

**Prototyper.**
Vill du testa en idé, bygga ett enkelt gränssnitt, eller skapa en liten intern app? Med AI i kodmiljön kan du komma från idé till något körbart på en förmiddag, utan att ha kodat tidigare.

---

## VS Code konkret

VS Code (Visual Studio Code) är en textredigerare för kod som kan installeras gratis. Den är ett populärt val för att jobba med AI-stöd, och det finns flera välbyggda tillägg:

**GitHub Copilot** skriver kod medan du skriver, föreslår nästa rad, och genererar funktioner baserat på kommentarer. Kräver prenumeration men är väl integrerat i VS Code.

**Continue** är ett öppet alternativ som kopplar till valfri AI-modell, inklusive Claude. Gratis och konfigurerbart.

Börja med att installera VS Code, lägg till ett av tilläggen ovan, och öppna en tom fil. Skriv en kommentar som beskriver vad du vill att koden ska göra. Tillägget genererar ett förslag.

---

## Claude Code konkret

Claude Code är ett terminalbaserat verktyg som arbetar direkt med filsystemet. Det kan:

- Läsa en hel kodbas och svara på frågor om den
- Skriva och ändra filer direkt
- Köra kommandon och tolka resultaten
- Arbeta iterativt på ett projekt utan att du klistrar in kod manuellt

Det är mer kraftfullt än ett VS Code-tillägg, men kräver lite mer initial konfiguration. Passar bra för dig som vill ge AI mer frihet att navigera och redigera i ett projekt.

Installation: `npm install -g @anthropic-ai/claude-code` (kräver Node.js).

---

## Vad du behöver för att komma igång

Du behöver inte kunna koda för att börja. Men du behöver:

- En konkret uppgift att lösa, inte "lär mig programmera"
- Förmåga att testa om det faktiskt fungerar
- Vilja att iterera när det inte fungerar direkt

Det sista är viktigt. Kod som AI genererar fungerar inte alltid på första försöket. Felsökning är en naturlig del av processen, och AI är bra på att hjälpa dig förstå vad som gick fel.

---

## Hur du kommer igång

1. Installera VS Code om du inte har det (code.visualstudio.com)
2. Lägg till GitHub Copilot eller Continue från Extensions-panelen
3. Beskriv en konkret uppgift i en kommentar i editorn
4. Låt AI generera ett förslag och läs igenom det för att förstå vad det gör
5. Kör det på ett testfall, inte på produktionsdata

---

## Labb

**Mål:** Bygg något litet med AI-stöd.

Välj en uppgift du gör manuellt och som följer ett upprepbart mönster.

1. Installera VS Code och lägg till ett AI-tillägg
2. Beskriv uppgiften på vanlig svenska i en kommentar i editorn
3. Låt AI generera ett script och läs igenom vad det gör
4. Kör det på ett testfall
5. Iterera: vad fungerade, vad behöver justeras?

Alternativt: installera Claude Code och ge det en liten uppgift som kräver att det läser och redigerar en fil.

---

## Sammanfattning

- Kodverktyg med AI-stöd är inte bara för utvecklare; de är för alla som vill automatisera och bygga
- VS Code med GitHub Copilot eller Continue ger AI-stöd direkt i kodmiljön
- Claude Code arbetar terminalbaserat med direkt filaccess och passar mer komplexa projekt
- Börja med en konkret, avgränsad uppgift och iterera

---

## Kom igång

- Installera VS Code om du inte har det
- Lägg till GitHub Copilot eller Continue
- Välj en manuell uppgift du vill automatisera
- Beskriv den i en kommentar och se vad AI genererar
