# Prompts

Den här mappen är till för återanvändbara prompts som kan hjälpa vid utveckling, granskning och dokumentation av AI Academy.

## Struktur

- `local/` används för lokala eller experimentella prompts och ignoreras av Git.
- Delade prompts kan läggas direkt i denna mapp eller i en namngiven undermapp.

## Regler

Skriv inte in:
- API-nycklar
- tokens
- lösenord
- privata länkar
- persondata
- interna åtkomstuppgifter

Håll prompts korta, tydliga och kopplade till ett konkret arbetsflöde.
