# Skapa ett konto på GitHub

GitHub är platsen där kod lever. Det är också där du hittar en stor del av de verktyg, projekt och resurser som driver AI-utvecklingen framåt.

Gå till github.com och skapa ett gratis konto. Sök sedan efter ett projekt eller verktyg du hört talas om — ett AI-ramverk, ett open source-projekt, eller något annat du är nyfiken på. Titta på hur det är strukturerat: koden, dokumentationen, historiken över ändringar.

Du behöver inte förstå allt. Det räcker med att börja orientera dig.
